# Contributing to OverTime

Thank you for your interest in OverTime! This is a personal homelab and learning
project, not a production tool. Contributions and feedback are appreciated with
that context in mind.

## Reporting Issues

Bug reports and feature requests are welcome. Please use the
[issue templates](https://github.com/jacob-stout/OverTime/issues/new/choose)
to submit them.

When reporting bugs, please include:

- Your operating system and Python version
- The scenario you were using (e.g., `ad-lab-m`, `k8s-dev`)
- The provider (`proxmox` or `azure`)
- Relevant log output, **with secrets redacted**

## Pull Requests

This project is not currently accepting pull requests. If you have an idea for
a change, please open an issue to discuss it instead.
